### 字符串反转
实现把'hello'转变成'olleh'
- 解法一：字符串和数组之间的相互转换，利用数组的reverse倒序方法
```
function strInversion(str) {
  if (typeof str !== 'string') {
    throw new TypeError(`${str} is not a string`);
  }
  return str.split('').reverse().join('');
}
```
- 解法二：利用数组的reduce方法实现
```
function strInversion(str) {
  if (typeof str !== 'string') {
    throw new TypeError(`${str} is not a string`);
  }
  return str.split('').reduce((prev, next) => {
    return next + prev;
  })
}
```
